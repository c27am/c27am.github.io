ALEXA'S FINAL PROJECT

I chose to make this project because gender inequality was always something that intrested me.

I was partly inspired by some of my community service to learn more about gender inequality in workplaces.

I used javascript and css to code ways to repersent data that made it easy for people to understand.

For my first visual, I repersentated data about how much women athletes are paid verses men athletes. In this code, I used "document.CreateElement" to create divs. Previously in the code I made a class for women and a class for man, which made pink circles for women and blue circles for men. Using this class I created divs using the class styling. I then used "var" to create a certian amount of women divs and a certian amount of men divs. Using classlist helped me so that I did not have to make multiple seperate divs.
